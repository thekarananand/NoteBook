# Docker Compose

#### Deploy the Application
```bash
docker compose up [--build / --no-build] [-d]
```
here, 
- `[--build]` -> Build images before starting containers
- `[--no-build]` -> Don't build an image
- `[-d]` -> detach mode
#### Take down the Application
```bash
docker compose down
```
#### List down the services
```bash
docker-compose ps
```
